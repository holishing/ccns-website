# ccns-website

Website Frontend for CCNS club.

## Getting Started

### Prerequisites

- [node](https://nodejs.org/en/)
- [yarn](https://yarnpkg.com/)

### Installing

Install packages

```bash
yarn install --prod=false
```


## Deployment

Initialize a dev server listened on `localhost:3000`

```bash
yarn run dev
```

## Production

Export static file in `dist` folder.

```bash
yarn run build
```
